# scriptstick
![image of scriptstick](https://github.com/axellse/crackedbadusb/raw/refs/heads/main/media/poster.png)

The scriptstick is a versatile scripting device in the form of a usb flash drive. It can be used for BadUsb, automation and meshtastic. The firmware writen in go, and custom scripts are written in lua. 

The scriptstick project is sponsored by [Hack club](https://hack.club).
